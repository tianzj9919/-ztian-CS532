# Sorting `listings.csv` by `host_name` and `price` 

## Build
```bash
gcc -Wall -Wextra -O2 listing.c -o listing
```

## Run
```bash
./listing listings.csv
```
This produces two files in the same directory:
- `sorted_by_host_name.csv`
- `sorted_by_price.csv`
